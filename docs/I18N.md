# Работа с локалями (i18n)

## Структура

- **Схемы**: `lib/i18n/schemas/` — источник истины для всех ключей
- **Локали**: `i18n/locales/{en,ru,kk,tr}/` — JSON файлы с переводами

## Добавление нового ключа

1. Добавить ключ в соответствующую схему в `lib/i18n/schemas/`
2. Добавить перевод во **все** локали в `i18n/locales/`
3. Запустить проверку:
   ```bash
   npx tsx scripts/check-i18n.ts
   ```

## Удаление ключа

1. Удалить ключ из схемы в `lib/i18n/schemas/`
2. Удалить из **всех** локалей в `i18n/locales/`
3. Запустить проверку:
   ```bash
   npx tsx scripts/check-i18n.ts
   ```

## Проверка консистентности

```bash
# Полная проверка (неиспользуемые ключи + консистентность)
npx tsx scripts/check-i18n.ts

# Только неиспользуемые ключи
npx tsx scripts/check-i18n.ts --unused

# Только консистентность между локалями
npx tsx scripts/check-i18n.ts --consistency

# Удалить неиспользуемые ключи
npx tsx scripts/check-i18n.ts --remove

# Удалить лишние ключи (не из en)
npx tsx scripts/check-i18n.ts --remove-extra
```
